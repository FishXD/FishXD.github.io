下载与安装

下载安装MySQL

可直接各大网站搜索即可

可根据https://www.jb51.net/article/177027.htm来进行操作

遇到的问题：

在此处提示3306端口被占用。

解决方法：

  1. 使用命令行cmd输入：

  1. 查看到进程的pid和其使用端口的情况

  2. 记住这个pid， 在任务管理器中结束该pid所代表的进程

  3. 在刚刚的上图界面中（MySQL的安装程序中）点击back，然后点击next回到上图界面就能发现端口已经不再被占用（即，刷新一下页面）

下载与安装navicat

因为我是以前疫情期间(2020年)学习需要下载安装过navicat并且使用了一段时间（增删改查的练习），所以这次就没有教程。

navicat连接MySQl

遇到的问题:

出现1251报错

1251 Client does not support authentication protocol requested by server

解决方法：

  1. cmd用管理员身份进入，然后输入
  1. newpassword就是新的密码

  1. 输入新密码

  2. 输入命令启动mysql

  3. 这里我遇到了新的问题：

    cmd提示ERROR 1045 (28000)
    解决方法：

    第一步 停止Mysql服务 ：

    停止 Mysql 服务，停止命令为：net stop mysql ，启动命令为：net start mysql 。

    这里又遇到了问题：

    cmd提示：net stop mysql服务名无效。
    解决方法：
    1. 使用管理员打开cmd
    2. 查找到mysql的真正名字
      1. 在我的电脑->右键点击->管理
      2. 服务和应用程序->服务->在右边找到MySQL的服务->右键点击->属性->服务名称
      3. 这里我的服务名称是mysql80
    3. 在第一步打开的cmd中输入net stop mysql80
    4. 停止成功
    5. 输入net start mysql80重新启动

第二步 修改Mysql配置文件：

进入到 Mysql 的bin 目录下(C:\ProgramData\MySQL\MySQL Server 8.0)找到 Mysql 的配置文件。如下图：

​ 第三步 添加配置信息skip-grant-tables

​ 打开文件，然后添加 skip-grant-tables （很重要）

​ 第四步 重新启动Mysql

​ 保存后在原来的CMD 窗口启动 Mysql ， 启动命令为：net start mysql80 。

​ 进入到 mysql 的安装目录，我的是： E:\mysql-5.6.35-winx64 。

​ 再输入： mysql -u root -p ，然后输入你的密码。然后回车。应该就提示成功了。
