idea learn

查找

编辑器基础搜索操作最有用的快捷方式之一是“查找操作”。它使您可以搜索所有可用操作，而不必知道它们的单独快捷方式。

Ctrl + Shift + A查找操作。

扩大和缩小代码选择

将插入符号放在任何单词之前。

Ctrl + Shift +向右箭头可移动插入符号到下一个单词，然后选择所有内容。

按Ctrl + W扩展选择到下一个代码块。

Ctrl + Shift +W缩小代码选择

复制和删除行

用Ctrl + D复制任何行。

您也可以多行执行相同的操作。只需选择两行或更多行，然后使用Ctrl + D复制它们。

要删除当前行，可以使用操作Ctrl + Y。

整行移动

重新排列行通常需要执行两个操作：剪切和粘贴。在此IDE中，您只需执行一个操作即可。

按Alt + Shift +向下箭头可以拉下当前行。

使用Ctrl + Shift +向上箭头向上移动整个方法。

折叠与放开代码块

有时您需要折叠一段代码以提高可读性。尝试使用Ctrl + NumPad-折叠代码片段。

要扩展代码区域，请按Ctrl + NumPad +

如果要折叠文件中的所有区域，请使用Ctrl + Shift + NumPad-。
同样，按Ctrl + Shift + NumPad +展开所有可用区域。

环绕和展开

按下Ctrl + Alt + T组合所选的代码片段和一些模板代码。

选择try/catch/finally。

让我们通过Ctrl + Shift + Delete展开操作返回到初始文件

选择展开尝试项。

多选

按Alt + J选择插入符号处的符号。

按Alt + J再次选择该符号的下一个出现。

按Alt + Shift + J取消选择最后一次出现。

按Ctrl + Alt + Shift + J选择文件中的所有匹配项。

键入td将所有出现的th替换为td。

智能完成

默认情况下，IntelliJ IDEA立即完成您的代码。在插入记号所在的位置开始右击Ran，您将看到带有匹配建议的Lookup菜单。您可以按Enter从Lookup菜单中选择第一项。

要激活基本完成功能，请按Ctrl +空格，您将再次看到查找菜单。

这里有问题： 微软自带的输入法快捷键于idea的ctrl + 空格冲突

解决方式：

按住win+空格更换为eng输入法

这样就可以解决

在查找菜单中选择i，然后按Enter。
按Ctrl + Shift + Enter完成此语句。

有时您需要查看有关静态常量或方法的建议。按两次Ctrl +空格可访问更深层次的代码完成。

智能填写

Smart Type Completion筛选建议列表，使其仅包含适用于当前上下文的那些类型。

按Ctrl + Shift +空格以查看匹配建议的列表。按Enter选择第一个

Smart Type Completion还可以为return语句建议代码。按Ctrl + Shift +空格两次以查看returm的Lookup菜单。按Enter选择第一个

后缀完成

后缀完成有助于减少编写代码时向后插入符号的跳跃。它使您可以根据添加的后缀，表达式的类型及其上下文，将已经键入的表达式转换为另一个表达式。

类型.在括号后面可以看到后缀完成建议的列表。从列表中选择是，还是在编辑器中键入它，然后按Enter键以完成语句。

声明完成

按Ctrl + Shift + Enter完成for语句。

编写if并按Ctrl + Shift + Enter生成语句。

在括号i％j == 0内添加一个条件，然后按Ctrl + Shift + Enter可以在if语句内跳转。

写在一行上：isPrime = false; break，然后按Ctrl + Shift + Enter来完成输入的语句并应用格式。

覆盖插入单词

按Ctrl +空格以显示完成选项。

例如，选择DO_NOTHING_ON_CLOSE，然后按Tab键。这将覆盖插入单词，而不是简单插入单词。

重命名

您可以通过将插入符号放在符号上并使用“重命名”重构来重命名任何内容。

将插入号放在代码中任何位置的deviceName字段上，然后按Shift + F6。

在此字段中输入新名称。例如，键入tabletName，然后按Enter保存结果。

重命名字段时，IntelliJ IDEA会自动检测匹配的getter方法，并建议对其进行重构。将插入号放置在手写笔字段上，并开始使用Shift + F6重命名。

例如，用pencil替换stylus，然后按Enter键完成重构。

提取变量/字段

按Ctrl + Alt + V提取变量或字段。

提取方法

提取方法使您可以将代码选择封装到具有可自定义参数和返回类型的新方法中。让我们将一些代码行提取到一个名为init（Dimension myDimension）的新方法中。按Ctrl + Alt + M。

重构基础

按Ctrl + Alt + C将选定的表达式提取为常量。

按Ctrl + Alt + M将选定的代码块提取到方法中。

按Ctrl + Alt + P将选定的表达式提取到参数中。

按Ctrl + Alt + V并从建议列表中选择新的BufferedReader（…），将其提取到变量中。

参数信息

按Ctrl + P查看方法签名。

快速弹出

按Ctrl + Q可以在插入符号处查看该符号的文档。

按Esc关闭弹出窗口。

按Ctrl + Shift + I可以在插入符号处查看符号的定义。

编辑器编码协助

按F2转到文件中下一个突出显示的错误。

按Ctrl + F1查看错误说明。

按Alt + Enter并选择“将例外添加到方法签名”。

按Ctrl + Alt + T并选择try / finally，将其与所选内容一起包围。

在fileReader的finally块中编写fileReader.close（）

按Ctrl + Shift + F7突出显示文件内插入符号处符号的所有用法。

跳转到源码

IntelliJ IDEA提供了多种导航功能。

让我们从最常用的功能之一“跳转到源代码”开始。按F4查找参数的来源。

您也可以跳转到JDK中实现的类的源。将插入符号放在String上，然后按F4。

声明/执行

使用Ctrl + B跳转到类或接口的声明。

要查看类/接口的实现，请使用Ctrl + Alt + B。

档案结构

在大型资源中导航非常复杂，有时您需要文件概览。使用Ctrl + F12查看文件结构。

下一个/上一个出现

IntelliJ IDEA允许您在事件之间进行导航。首先，在代码中选择手机，然后按Ctrl + F调用“查找”操作。

要查找下一次出现，请使用Enter或F3

对于下一出现出现，请按Shift + F3

关闭搜索面板后，您仍然可以使用这些快捷方式在之前搜索的项目之间进行导航。按Esc键关闭搜索面板，然后尝试使用F3查找下一个事件

然后向后移动Shift + F3
